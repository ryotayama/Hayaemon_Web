聞々ハヤえもん ( ぶんぶんはやえもん ) は、再生速度・再生周波数・音程を自由に変更できるMP3プレーヤとして公開中のオープンソースのフリーソフトです。
WAVE, MP3, Ogg Vorbis, WMA などの読み込みに対応しており、再生速度・再生周波数・音程のそれぞれ独立した変更が可能なMP3プレーヤです。

スクリーンショット




ここに置いてある画像は、聞々ハヤえもんの紹介などの用途に、自由にお使いいただくことができます。

●アイコン

聞々ハヤえもんのアイコンです。
ハヤえもん ( Hayaemon ) の頭文字である Ｈ ( エイチ ) を音符のマーク風にデザインしています。

●スクリーンショット その１

再生リストとコントロールバーをすべて閉じた状態です。
コントロールバーを閉じたからといって、再生速度や再生周波数・音程がリセットされるわけではなく、それまでに変更した再生速度や音程などの変更はすべて有効です。
再生リストやコントロールバーの開閉は、ツールバー上から行うことができます。
また、１曲ループ、全曲ループ、ランダム再生を、ツールバー上から切り替えることができます。

●スクリーンショット その２

コントロールバーを開いた状態です。
このコントロールバーにより、再生速度や再生周波数・音程を、それぞれ独立して変更することができます。

●スクリーンショット その３

再生リスト、コントロールバー、グラフィックイコライザをすべて開いた状態です。
２つめのコントロールバーでは音量とパンを、グラフィックイコライザでは周波数帯域ごとのゲインを変更することができます。
